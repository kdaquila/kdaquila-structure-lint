# kdaquila-structure-lint

Opinionated Python project structure and code quality linter.

## Status

This package is currently in development (Phase 1: Repository Setup).

## Overview

`kdaquila-structure-lint` provides automated validation for Python projects with a focus on:

- Line limit enforcement (default: 150 lines per file)
- One function/class per file validation
- Opinionated folder structure validation (opt-in)

## Installation

```bash
pip install kdaquila-structure-lint
```

## Usage

```bash
structure-lint
```
